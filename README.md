# Zhonghe Village InSAR 3D Slip Prediction Preview

This is a static 3D preview of preliminary InSAR-predicted slope slip zones for Zhonghe Village, Shifang Town, Wuping County, Fujian, China.

Open the GitHub Pages site to view the interactive terrain overlay.

## Layers

- Satellite imagery texture draped on DEM terrain
- Preliminary InSAR slip prediction zones
- Confidence colors:
  - Red: high confidence
  - Orange: medium confidence
  - Yellow: low confidence

## Important Note

This is a preliminary InSAR screening and visualization product, not a field-confirmed landslide inventory or surveyed hazard boundary.
