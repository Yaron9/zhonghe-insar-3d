# Zhonghe Village InSAR 3D Slip Prediction Preview

Interactive 10 km x 10 km 3D preview of preliminary InSAR-predicted slope slip zones for Zhonghe Village, Shifang Town, Wuping County, Fujian, China.

GitHub Pages:

https://yaron9.github.io/zhonghe-insar-3d/

## Current Version

- AOI: 10 km x 10 km
- Terrain: DEM mesh, 229 x 227 grid, 2.0x vertical exaggeration
- Imagery: Esri World Imagery z15, external 4096 px texture
- Prediction zones: 86 zones, 3131 pixels
- LOS-visible candidate pixels: 1761, about 56.2%

## Confidence Colors

- Red: high confidence, 1 zone / 23 pixels
- Orange: medium confidence, 39 zones / 1658 pixels
- Yellow: low confidence, 46 zones / 1450 pixels

## Important Note

This is a preliminary InSAR screening and visualization product, not a field-confirmed landslide inventory or surveyed hazard boundary.
